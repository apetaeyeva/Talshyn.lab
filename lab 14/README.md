Краткое описание реализованных функций

5.1. calculate_average(grades)

Осуществляет:
	•	проверку корректности входных данных,
	•	проверку типа элементов,
	•	расчёт среднего значения,
	•	фиксацию ошибок и успешных операций в лог-файл.

5.2. determine_grade_letter(avg)

Преобразует числовое значение среднего балла в буквенную оценку на основе стандартной шкалы.

5.3. student_report(name, grades)

Формирует итоговый текстовый отчёт на основе имени студента и списка оценок. Производит логирование создания отчёта.

6. Unit-тестирование

Тестирование реализовано с использованием стандартного модуля unittest.

Проверяемые аспекты:
	•	корректность вычисления среднего значения;
	•	обработка некорректных данных;
	•	соответствие буквенной оценки заданной шкале;
	•	корректность формирования итогового отчёта.
